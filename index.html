HTML Tags Syntax:

<element-name>Content</element-name>

A single tag is made of an opening angle bracket <, followed by the element-name, followed by the closing angle bracket >.

Self Closing Tags:

<self-closing-element-tag />

While most elements are made of opening and closing tags some are self-closing. Furthermore, some self-closing elements like <br> can have a slash at the end like so: <br>. The other self-closing tags include: <area>, <base>, <col>, <embed>, <hr>, <img>, <input>, <keygen>, <link>, <meta>, <param>, <source>, <track>, and <wbr>.

Form Tags:

In HTML, forms are used for gathering and sending user input to an external source (like a server) for processing. This is done with the <form> element, which contains various labels and inputs.

<form>
 <!-- Form elements -->
</form>

The following is an example of an HTML form that features <input> elements:

<!DOCTYPE html>
 <html>
  <head> </head>
  <body>
   <form method="post" action="http://server1">
    Enter your name:
    <input type="text" name="fname" />
    <br />
    Enter your age:
    <input type="text" name="age" />
    <br />
    <input type="submit" value="Submit" />
   </form>
  </body>
 </html>

Table Tags:

<table>  //table: Defines an HTML table.
  <thead> //thead: Groups the header content in a table.
    <tr> //tr: Defines a row in a table.
      <th>heading 1</th> //th: Defines a header cell in a table.
      <th>heading 2</th>
      <th>heading 3</th>
    </tr>
  </thead>
  <tbody> //tbody: Groups the body content in a table.
    <tr>
      <td>cell 1</td> //td: Defines a cell in a table that contains data.
      <td rowspan='2'>cell 2</td>
      // rowspan: Specifies the number of rows a cell should span.
      <td>cell 3</td>
    </tr>
    <tr>
      <td>cell 1</td>
      <td>cell 3</td>
    </tr>
    <tr>
      <td>cell 1</td>
      <td colspan='2'>cell 2+3</td>
    </tr>  //colspan: Specifies the number of columns a cell should span.
  </tbody>
  <tfoot> //tfoot: Groups the footer content in a table.
    <tr>
      <td>foot 1</td>
      <td>foot 2</td>
      <td>foot 3</td>
    </tr>
  </tfoot>
</table>

The above code block shows the HTML markup for a simple table structure. Its output is as follows:

heading 1 heading 2 heading 3
cell 1 cell 2 cell 3
cell 1 cell 3
cell 1 cell 2+3
foot 1 foot 2 foot 3
